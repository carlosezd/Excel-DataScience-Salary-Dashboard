# Introducción

El presente proyecto es parte del desarrollo del curso de “Excel for Data Analytics” de Luke Barousse https://www.youtube.com/watch?v=pCJ15nGFgVg&list=PL_CkpxkuPiT-RJ7zBfHVWwgltEWIVwrwb para practicar los conceptos aprendidos de la primera parte del curso. 

El proyecto consiste en diseñar un dashboard utilizando el dataset sobre ofertas de empleos relacionados a data que fueron publicados en 2023. El dasbhoard tiene el propósito de dar información referencial sobre el salario en función del puesto, la modalidad del trabajo y el país donde se desarrolla dicho puesto. Y a diferencia del ejemplo del curso se hicieron algunas modificaciones en el formato para cumplir con algunas buenas prácticas de visualizaciones de datos.

# Dataset

Los datos fueron recopilados por Luke Barousse, a partir ofertas de empleos publicadas en diferentes plataformas como linkedin o ziprecruiter. Este archivo fue descargado desde la carpeta del curso https://github.com/lukebarousse/Excel_Data_Analytics_Course. Tiene columnas como:

- Título del puesto
- Tipo de horario del Empleo
- País del puesto
- Fecha de publicación de la oferta
- Job-Skills
